# AGENTS.md

## 项目说明

本仓库是“人教版高中物理·热力学第一定律”网页课件项目。

## 开发约束

- 使用原生 HTML、CSS 和 JavaScript。
- 不引入 React、Vue、Vite 等框架。
- 不安装依赖。
- 不调用外部图片、字体、CSS 或 JavaScript。
- 图形优先使用内联 SVG。
- 禁止使用 Canvas。
- 保持 16:9 课件比例，避免页面滚动条。

## 物理内容约定

统一采用：`ΔU = Q + W`。

- `Q > 0` 表示系统吸热。
- `Q < 0` 表示系统放热。
- `W > 0` 表示外界对系统做功。
- `W < 0` 表示系统对外界做功。

修改课件时不得混用其他正负号约定。
